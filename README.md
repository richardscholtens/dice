# Dice
Graphical dice


A program that designs six dice forming a straight from
left to right in order. If someone clicks on a die it
will show the given number of that die.


Assignment:

Write a program that draws 6 dice on the screen depicting each of the six values (1, 2, 3, 4, 5 and 6). If the user clicks on a die, the program should print the corresponding value. If the user clicks outside any die, the program should halt (exit, stop, quit).
